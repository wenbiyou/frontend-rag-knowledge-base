## 问题分析

通过检查代码，发现深色主题不工作的原因是 **Tailwind CSS 配置文件中缺少了 `darkMode` 配置选项**。

## 具体问题

在 `/frontend/tailwind.config.js` 文件中，没有设置 `darkMode` 配置，默认情况下 Tailwind CSS 的深色模式是禁用的。

## 修复方案

1. **修改 tailwind.config.js 文件**：
   - 添加 `darkMode: 'class'` 配置选项
   - 这样 Tailwind CSS 就会根据 `dark` 类名来应用深色模式样式

## 修复原理

- ThemeProvider 组件已经正确实现了在深色模式下为 `document.documentElement` 添加 `dark` 类的功能
- 但由于 Tailwind CSS 没有启用深色模式支持，所以即使添加了 `dark` 类，深色模式样式也不会被应用
- 通过在 Tailwind 配置中添加 `darkMode: 'class'`，可以启用基于类名的深色模式支持

## 预期效果

修复后，用户应该能够：
1. 通过主题切换按钮切换浅色/深色模式
2. 选择跟随系统主题
3. 在深色模式下看到正确的深色主题效果
