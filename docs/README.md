# 前端知识库 AI 助手 - 文档中心

欢迎使用前端知识库 AI 助手！这里汇集了所有相关文档。

## 系统架构概览

```
┌─────────────────────────────────────────────────────────────────┐
│                    前端层 (Next.js 14)                           │
│   首页问答 │ 管理后台 │ 登录认证 │ 主题切换 │ 会话管理            │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    API 网关层 (FastAPI)                          │
│   认证 │ 对话 │ 文档 │ 同步 │ 管理 │ AI功能 │ 社区              │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    服务层 (Business Logic)                       │
│   RAG引擎 │ AI导师 │ 同步服务 │ 推荐 │ 知识图谱 │ 代码分析       │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────┐
│                    数据层                                        │
│   ChromaDB(向量) │ SQLite(关系) │ DeepSeek(LLM) │ GitHub API    │
└─────────────────────────────────────────────────────────────────┘
```

## 功能模块索引

### 核心功能

| 模块     | 说明                 | 相关 API                        |
| -------- | -------------------- | ------------------------------- |
| 智能问答 | 基于 RAG 的智能对话  | `/api/chat`, `/api/chat/stream` |
| 文档管理 | 文档上传、查看、删除 | `/api/documents`, `/api/upload` |
| 会话管理 | 对话历史持久化       | `/api/sessions`                 |
| 搜索建议 | 输入时智能提示       | `/api/suggestions`              |

### 同步功能

| 模块         | 说明                            | 相关 API                               |
| ------------ | ------------------------------- | -------------------------------------- |
| 官方文档同步 | React/Vue/TypeScript 等官方文档 | `/api/sync`                            |
| GitHub 同步  | 仓库文档自动同步                | `/api/repos`, `/api/webhook/github`    |
| 云端同步     | 数据导出/导入                   | `/api/sync/export`, `/api/sync/import` |

### AI 增强功能

| 模块     | 说明               | 相关 API                                        |
| -------- | ------------------ | ----------------------------------------------- |
| RAG 优化 | 查询优化、重排序   | `/api/rag/optimize`, `/api/rag/enhanced-search` |
| AI 导师  | 学习计划、技能评估 | `/api/mentor/*`                                 |
| 知识图谱 | 技术关联可视化     | `/api/graph/*`                                  |
| 智能推荐 | 个性化内容推荐     | `/api/recommendations/*`                        |
| 代码分析 | 代码文件分析       | `/api/code/*`                                   |

### 管理功能

| 模块         | 说明              | 相关 API           |
| ------------ | ----------------- | ------------------ |
| 用户认证     | 登录、注册、权限  | `/api/auth/*`      |
| API Key 管理 | 开放 API 密钥管理 | `/api/keys`        |
| 统计分析     | 使用数据统计      | `/api/analytics/*` |
| 反馈系统     | 点赞、错误标记    | `/api/feedback/*`  |

### 社区功能

| 模块       | 说明           | 相关 API                   |
| ---------- | -------------- | -------------------------- |
| 提示词分享 | 优质提示词共享 | `/api/community/prompts`   |
| 配置分享   | 知识库配置共享 | `/api/community/configs`   |
| 最佳实践   | 经验案例分享   | `/api/community/practices` |
